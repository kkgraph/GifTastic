# GifTastic
App: idk my bff gif?
Using common abbreviations used in texts, you can generate gifs from GIPHY using the GIPHY API. You can also add your own gif by adding a new button.

## Built With

* HTML5
* CSS
* jQUERY
* Javascript 
* API/AJAX 
* Bootstraps
